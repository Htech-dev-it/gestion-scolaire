# Guide du Portail Élève

Bienvenue sur votre portail personnel. Ce guide vous explique comment utiliser toutes les fonctionnalités à votre disposition pour suivre votre scolarité.

## Tableau de Bord

Votre tableau de bord est la page d'accueil de votre portail. Il vous donne un accès rapide à tout ce dont vous avez besoin.

### Aperçu
C'est la section principale de votre tableau de bord. Vous y trouverez des cartes de navigation pour accéder rapidement aux différentes sections de votre portail :
- **Emploi du temps**: Pour voir votre horaire de cours.
- **Mes Notes**: Pour consulter vos résultats académiques.
- **Ressources**: Pour accéder aux documents partagés par vos professeurs.
- **Documentation**: Pour consulter ce guide d'utilisation.

### Sécurité
L'onglet "Sécurité" est très important. Il vous permet de changer votre mot de passe.
1.  Entrez le mot de passe actuel (celui que l'administration vous a donné).
2.  Entrez votre nouveau mot de passe personnel.
3.  Confirmez votre nouveau mot de passe.
4.  Cliquez sur "Changer le mot de passe".

**Il est très important de changer votre mot de passe lors de votre première connexion.**

## Emploi du temps

Cette page affiche votre horaire de cours pour la semaine. Vous pouvez y voir :
- Le nom de la matière.
- L'heure de début et de fin du cours.
- Le nom du professeur.
- La salle de classe.

L'emploi du temps est mis à jour automatiquement par l'administration.

## Mes Notes

C'est ici que vous pouvez suivre vos performances académiques.
1.  **Choisissez une période**: Utilisez les onglets en haut (ex: "Trimestre 1", "Trimestre 2") pour voir les notes de cette période.
2.  **Résumé de la période**: Vous verrez votre moyenne générale pour la période et une appréciation globale de vos professeurs.
3.  **Détail par matière**: Pour chaque matière, vous trouverez :
    - Votre moyenne dans cette matière.
    - Une appréciation de votre professeur pour cette matière.
    - La liste de toutes vos notes (devoirs, interrogations, examens).

## Ressources

Cette page centralise tous les documents et liens partagés par vos professeurs.
- Les ressources sont regroupées par matière pour que vous puissiez les retrouver facilement.
- Cliquez sur une ressource pour la consulter.
  - Les **PDF** et les **images** s'afficheront directement dans une fenêtre.
  - Les **vidéos YouTube** pourront être lues sur place.
  - Les **liens web** s'ouvriront dans un nouvel onglet.
- Un bouton "Télécharger" est disponible si vous souhaitez enregistrer une copie d'un fichier sur votre ordinateur.
